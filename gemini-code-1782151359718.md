# Project 2: Automated Sliding-Window Approach for Pan-Flavivirus Diagnostic Primer Design

## 1. Executive Summary
This project outlines the development of an automated Python-based bioinformatics pipeline designed to uncover highly conserved nucleotide regions across multiple viral genomes. Using a sliding-window algorithm, the tool successfully evaluated Dengue virus strains circulating in Sri Lanka to engineer an optimized diagnostic primer candidate. In silico validation via NCBI BLAST confirmed that the engineered candidate functions as a universal pan-flavivirus primer, achieving 100% identity matches across Dengue Serotypes 1–4 and Zika virus.

## 2. Background & Objectives
Designing effective molecular diagnostics (like RT-PCR assays) for RNA viruses is challenging due to high mutation rates and spatiotemporal genomic diversity. Traditional alignment methods can be computationally intensive. This project aimed to:
* Build an efficient pipeline utilizing a sliding-window assay to find conserved regions without requiring exhaustive full-genome alignments.
* Target specific molecular patterns from regional field isolates (Sri Lanka).
* Filter candidates based on clinical constraints (GC content and Melting Temperature).

## 3. Methodology & Computational Pipeline
The project was executed through the following pipeline phases:
1. **Data Acquisition:** Obtained complete genome sequences of Dengue Virus Type 1 and Type 2 isolating geographic location parameters restricted to Sri Lanka from the NCBI Virus Database.
2. **Data Cleansing & Aggregation:** Consolidated separate serotype datasets into a singular multi-FASTA master repository (`dengue_sequences.fasta`).
3. **Sliding-Window Optimization:** Developed `designer.py` utilizing Biopython. The script scans the template genome 20bp at a time, evaluating the exact percentage representation of each slice across the entire dataset.
4. **Biochemical Filtering:** Screened windows against strict clinical parameters:
   * **GC Content:** Restricted to 40%–60% for structural stability.
   * **Melting Temperature ($T_m$):** Target window adjusted between 52°C and 60°C (using Wallace calculation criteria).

## 4. Results & Outputs
The pipeline evaluated 8 complete genomes and identified 1,827 conserved target regions meeting initial selection thresholds. The top-performing computed diagnostic sequence was:

* **Sequence (5' -> 3'):** `AAACAGCATATTGACGCTGG`
* **Genome Alignment Position:** Index 10626
* **Dataset Conservation Rate:** 88%
* **GC Content:** 45.0%
* **Melting Temperature ($T_m$):** 58.0°C

## 5. In Silico Validation (NCBI BLAST Results)
Validation of the candidate primer against the global GenBank database returned exceptional results:
* **Target Breadth:** 100% Identity and 100% Query Coverage against Dengue Serotypes 1, 2, 3, and 4.
* **Cross-Reactivity Detection:** Identified 100% identity matching with Zika virus isolates, highlighting an alternative cross-reactive application as a broad-spectrum pan-flavivirus diagnostic tool.
